# Плагин полей RadicalMart Field Subform для CMS Joomla

### 1. Что это?
RadicalMart Field Subform - плагин конструктора повторяемого поля для компонента RadicalMart.

### 2. Версии Joomla
Плагин для версий Joomla 3.9+, 4+.
